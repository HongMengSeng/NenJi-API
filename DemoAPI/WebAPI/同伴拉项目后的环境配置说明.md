- 找不到 `Microsoft.AspNetCore.Authentication.JwtBearer`

## 8. 常见报错处理

### 8.1 报错：找不到 `Microsoft.AspNetCore.Authentication.JwtBearer`

处理方法：

1. 先执行：

```powershell
dotnet restore
```

2. 再执行：

```powershell
dotnet build
```

3. 如果还不行，检查：

- [NuGet.Config](/C:/Users/Administrator/Desktop/WebAPI/NuGet.Config) 是否存在
- 是否能访问 `https://api.nuget.org/v3/index.json`

### 8.2 报错：数据库连接失败

常见原因：

- MySQL 没启动
- 数据库 `Test3` 不存在
- 用户名密码不对
- 本机端口不是 `3306`

处理方式：

- 检查 [appsettings.json](/C:/Users/Administrator/Desktop/WebAPI/WebAPI/appsettings.json)
- 用数据库客户端先验证 `root/root` 是否能登录

### 8.3 报错：小程序请求 `ERR_CONNECTION_REFUSED`

常见原因：

- 后端没启动
- 前端请求端口错了
- `api.js` 基础地址不对

处理方式：

- 先确认后端启动成功
- 再检查 [api.js](/C:/Users/Administrator/Desktop/WebAPI/demo/utils/api.js)

### 8.4 报错：真机不能访问 localhost

说明：

- 微信开发者工具里可以用 `localhost`
- 真机调试时不能用 `localhost`

处理方式：

- 把接口地址改成电脑局域网 IP
- 在微信小程序后台配置合法请求域名
